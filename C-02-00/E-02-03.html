Basic Structure of an HTML5 Document

In an HTML document, the basic structure includes the following elements:

HTML Basic Structure
  • <html> - Root element
  • <head> - Contains metadata, links to stylesheets, etc.
  • <body> - Contains the content of the webpage

HTML Elements and Components

HTML Elements

HTML elements are the building blocks of an HTML page. Common elements include:

  • <p> - Paragraph
  • <a> - Anchor (hyperlink)
  • <img> - Image
  • <div> - Division or section
  • <h1> to <h6> - Headings

HTML5 Semantic Elements

Semantic Elements

HTML5 introduces semantic elements that improve the structure and accessibility of web pages:

  • <header> - Header section
  • <footer> - Footer section
  • <article> - Independent content
  • <section> - Section of content
  • <nav> - Navigation links
